# CHECK API 시장명 ↔ 코드(apiurl) 룩업

공식 API 일람(https://checkapi.koscom.co.kr/intro)의 **시장명**을, 실제 apiurl에 쓰는
**패밀리 코드(`mXXX`)·도메인**으로 바로 찾기 위한 표다. `domains/*.md`와 `endpoint-catalog.md`는
`mXXX` 코드로만 구성돼 있어 "국채30년 선물"·"컨센서스"·"가상자산" 같은 **시장명으로는 못 찾는다**
— 이 문서가 그 룩업을 채운다.

> 출처: `checkapi-specs.json`의 `menus[]`(ground truth, 이름 필드는 cp949 인코딩).
> 지수·KONEX·도메인경계 항목은 live `code_info` 실측으로 교차검증(2026-07).
>
> ⚙ **이 표는 런타임 소스 `.claude/skills/checkapi-data/scripts/_common.py`의 `MARKET_NAMES`와
> 동기화돼야 한다.** `MARKET_NAMES`에 패밀리를 추가했는데 이 문서에 반영을 빠뜨리면 **커밋이 막힌다**
> (pre-commit 드리프트 검사 `check_market_index.py`). 수동 실행: `python check_market_index.py`.

## ⚠ 먼저 — 도메인 경계 불일치 3건
웹 분류(메뉴)와 실제 apiurl 도메인이 다른 항목. **도메인을 틀리면 endpoint를 못 찾는다.**

| 시장명 | 웹 메뉴 분류 | 실제 apiurl 도메인 |
|---|---|---|
| **해외환율**(원/달러중개 외 위안/달러 등) | 채권-API | **`/ext/m174`** (라이센스 필요) |
| **가상자산** | 기타-API | **`/ext/m215`** (실측 299종: 제로지·1인치 등) |
| (참고) 해외금리 | 채권-API | `/bond/m025` (도메인 일치 — 예외 아님) |
