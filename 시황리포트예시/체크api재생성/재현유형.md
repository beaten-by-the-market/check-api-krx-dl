# CHECK API 시황 리포트 재현 — 유형 범례

이 폴더의 재생성 리포트에서 **CHECK API 기본 구독으로 채워지지 않은 값은 `[유형N]`** 으로 표기했다.
각 값이 왜 안 채워졌는지, 어떻게 하면 채울 수 있는지는 아래 유형 정의로 확인한다.
(유형 표기 없는 값 = CHECK API로 실측 검증된 재현값.)

## 요약

| 유형 | 의미 | 재현 여부 | 해결책 / 비용 |
|---|---|---|---|
| **유형1** | 추가 구독하면 채워짐 | **사실상 재현가능** | 해외시세 패키지 **월 80,000원** 등 |
| **유형2** | CHECK에 데이터 자체가 없음 | 재현 불가 | 외부 소스 필요 |
| **유형3** | 스냅샷/시점 불일치 | 값은 있으나 시점·세션이 달라 편차 | 그날 저녁 실행·기준 차이 감안 |
| **유형4** | 산식/외부기준 필요 | 직접 산출 불가 | 별도 공식·데이터 |

## 유형1 — 추가 구독 시 재현 가능 (사실상 재현가능)

견적서(2026-02-13, KOSCOM)의 '선택정보'를 계정에 월 추가하면 CHECK-API로 조회된다.
(CHECK-API 자체는 데이터량 요금제 Basic 100,000 ~ Premium3 5,000,000이고, 데이터 '종류'는 선택정보 구독으로 붙는다.)

| 항목 | 구독 | 월 비용 |
|---|---|---|
| **미국 다우존스·나스닥 지수** | **해외시세 패키지**(`ext/m184`) | **80,000원** |
| 실시간 해외 개별종목·해외환율 전용피드(`ext/m174`) | 해외시세 패키지 | (위 패키지 내) |

> ⚠ **정정(2026-07 실측)**: 예전엔 해외증시 지수·위안/달러 전체를 유형1(구독)로 봤으나,
> **경제지표(economic)에 라이센스 없이 대부분 있다**. **유형1로 남는 건 미국 다우존스·나스닥 뿐** —
> 나머지(S&P500·니케이·상하이·항셍·대만·DAX·유로스톡스·위안/달러·엔·유로)는 아래 "기본 구독 재현"으로 이동.

## 유형2 — CHECK 데이터 자체 부재 (구독으로도 불가)

- **투자주체별 공매도 비중**(외국인/기관/개인): CHECK의 공매도 데이터(거래·잔고)에 **투자자 차원이 아예 없음**(공매도 endpoint 3종·전 공매도 필드 전수 확인). → KRX 공매도종합포털 등 외부 필요.
- **국제유가 Brent(두바이 포함)**: CHECK 전체(경제지표·전 endpoint 명세)에 **전혀 없음**(원자재 지표는 WTI·휘발유뿐). Brent는 ICE 거래라 견적서 해외파생(CME/SGX/HKEX)에도 없음 → 외부 소스 필요. (WTI는 `USCOMCL1D`로 재현됨 — 아래 참고)

## 유형3 — 스냅샷/시점 불일치

값(또는 데이터)은 있으나 **어느 시점의 스냅샷이냐**가 리포트와 달라 정확히 안 맞는 것. 두 하위 형태:

- **날짜 시점 (당일 전용)**: 관련 endpoint에 날짜 파라미터가 없어 '당일 스냅샷'만 준다.
  → **그날 저녁 실행하면 정확 재현**되나 과거 특정일 소급은 안 됨.
  예: 종목동향 종목수(상승/하락/상한/하한/보합)·특징주(등락률 상위)·프로그램 차익/비차익 당일치.
- **세션 시점 (정규장 vs 전체)**: CHECK 거래대금(`F15023`)은 전체(시간외 포함) EOD값, KRX 리포트는 정규장 15:45
  기준이라 ~2% 차이. CHECK엔 **정규장 전용 거래대금 필드가 없어** KRX 시점과 정확히 못 맞춘다.
  예: 공매도 거래비중·시장별 공매도대금 비중(금액 자체는 근접, '비중'만 편차).

## 유형4 — 산식/외부기준 필요

- **KOSPI200 예상시가·괴리율**(장개시): 동시호가 선물 예상체결 기반 **시황팀 산식**이 필요(입력은 라이브로 확보 가능하나 변환 공식은 별도).

## 참고: 기본 구독으로 재현되는 매크로 지표

아래는 추가 구독 없이 CHECK API로 재현된다(리포트에서 유형 표기 없이 값이 채워진 것).

- **원/달러(spot)**: 외국환중개 `bond/m023 basic_info jcode=00USDSP` — F15001 현재가·F15183 당일매매기준율. (실측 1,525.6) ※ 당일 basic_info만 안정, 과거일 hist는 쿼리오류.
- **국내 국고채 3년/10년 지표금리(일별)**: 채권 지표물 — `bond/m058/jipyo_list edate=<날짜>` 로 만기별 지표물 ISIN(만기10년은 '국고…' 명목물) → `bond/m038/hist_info` **F15175 수익률**. `edate`로 로테이션 자동 해결. (실측 3년 3.745·10년 4.168)
- **US 10년 국채금리**: 채권 해외금리 `bond/m025/hist_info jcode=GBUS10Y` F32450 중간호가. (실측 4.486 = 시황 정확)
- **달러지수**: 경제지표 `etc/economic/indicator check_code=USDXYD`. (실측 100.87 = 시황 100.8 정확)
- **국제유가 WTI·휘발유선물**: 경제지표 `USCOMCL1D`(WTI, 68.76)·`USCOMXB1D`(휘발유선물, 2.95).
- **해외 주가지수(미국 다우·나스닥 제외 전부)**: 경제지표 일별 — `USSPX500D` S&P500(7,483)·`JPNIKKEI225D`
  니케이(69,744)·`CNSHCOMPD` 상하이(4,044)·`HKHSID` 항셍(23,350)·`TWTWSED` 대만가권(46,781)·`DEDAXD`
  DAX(25,779)·`EASTOXX50D` 유로스톡스50(6,413) + 인도·베트남·호주 등. `market_brief.py --overseas`로 자동.
- **환율 크로스**: `USDCNYTED` 위안(6.78)·`USDJPYTED` 엔(161.37)·`EURUSDTED` 유로 — **위안/달러도 라이센스 무관**(정정).
- **각국 기준금리**: `USFEDRATED` Fed(3.75)·`KRBOKRATED` 한은(2.5)·`EAECBRATED` ECB(2.40)·`JPBOJRATED` BOJ(1.0) 등 18개국.
- **미국 국채 커브**: `USUST3MRD`~`USUST30YRD`(3M 3.76·2Y 4.18·10Y 4.49·30Y 4.98) 전만기.
  ※ economic은 전체 히스토리 반환이라 느리고, 일부 만기(예 5Y)는 갱신 지연될 수 있어 날짜(TIME) 확인.
- 참고: 경제지표(30,316개)는 각국 금리·환율·지수·유가·GDP·CPI·고용 등 글로벌 매크로 방대. 실시간성은 일별(D) 1,297개.
