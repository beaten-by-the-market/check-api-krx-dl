# CHECK API 시황 리포트 재현 — 유형 범례

이 폴더의 재생성 리포트에서 **CHECK API로 채워지지 않은 값은 `[유형N]`** 으로 표기했다.
각 값이 왜 안 채워졌는지, 어떻게 하면 채울 수 있는지는 아래 유형 정의로 확인한다.

## 요약

| 유형 | 의미 | 재현 여부 | 해결책 / 비용 |
|---|---|---|---|
| **유형1** | 추가 구독하면 채워짐 | **사실상 재현가능** | 해외시세 패키지 **월 80,000원** 등 |
| **유형2** | CHECK에 데이터 자체가 없음 | 재현 불가 | 외부 소스 필요 |
| **유형3** | 당일 스냅샷만 제공 | 당일 실행 시 재현(과거일 불가) | 그날 저녁 배치 실행 |
| **유형4** | 근사치(값 편차) | 재현되나 값이 다름 | KRX와 집계기준 차이 |
| **유형5** | 산식/외부기준 필요 | 직접 산출 불가 | 별도 공식·데이터 |

> 나머지(유형 표기 없는 값)는 CHECK API 기본 구독으로 **실측 검증된 재현값**이다.

---

## 유형1 — 추가 구독 시 재현 가능 (사실상 재현가능)

견적서(2026-02-13, KOSCOM) 기준 아래 '선택정보'를 계정에 월 추가하면 CHECK-API로 조회된다.
(CHECK-API 자체는 데이터량 요금제 Basic 100,000 ~ Premium3 5,000,000이고, 데이터 '종류'는 선택정보 구독으로 계정에 붙는다.)

| 항목 | 구독 | 월 비용 |
|---|---|---|
| **해외증시 지수** — 니케이·상해종합·대만·홍콩HSI·다우·나스닥 | **해외시세 패키지** (아시아·미국·글로벌 지수) | **80,000원** |
| 유럽지수(DAX 등) | 글로벌지수(STOXX Europe 600) 포함 — DAX 정확 커버는 확인 필요 | (위 패키지 내) |
| **US 10년 국채금리** | 채권고급(해외금리 포함) | 50,000원 |
| **원/달러·위안/달러·달러지수(외환)** | 외환은 **기본 정보료에 이미 포함** | 추가 0원 (단 CHECK-API endpoint 노출 여부 확인 필요) |

→ 즉 해외증시·US금리 블록은 **월 8~13만원 추가로 사실상 재현 가능**. 외환은 이미 구독 중이라 API 노출만 확인하면 무료로 채워진다.

## 유형2 — CHECK 데이터 자체 부재 (구독으로도 불가)

- **투자주체별 공매도 비중**(외국인/기관/개인): CHECK의 공매도 데이터(거래·잔고)에 **투자자 차원이 아예 없음**(공매도 endpoint 3종·전 공매도 필드 전수 확인). → KRX 공매도종합포털 등 외부 필요.
- **국제유가 WTI/Brent**: 견적서 해외파생(CME/SGX/HKEX)에 원유선물이 명시되지 않음 → 별도 데이터 소스 필요.

## 유형3 — 당일 스냅샷 한계 (당일 실행이면 재현)

- **종목동향 종목수**(상승/하락/상한/하한/보합), **특징주(등락률 상위)**, **프로그램 차익/비차익 당일치**: 관련 endpoint가 날짜 파라미터 없는 '당일 스냅샷'. **그날 저녁 실행하면 정확 재현**되나 과거 특정일 소급은 안 됨.

## 유형4 — 근사치 (값 편차)

- **공매도 거래비중 / 시장별 공매도대금 비중**: CHECK 거래대금(F15023)과 KRX 총거래대금(정규장 기준)의 집계 정의가 달라 비중값이 ~2% 다름. CHECK엔 **정규장 전용 거래대금 필드가 없어** 안에서 KRX 기준과 정확히 맞출 수 없음. (공매도 '금액' 자체는 근접)

## 유형5 — 산식/외부기준 필요

- **KOSPI200 예상시가·괴리율**(장개시): 동시호가 선물 예상체결 기반 **시황팀 산식**이 필요(입력은 라이브로 확보 가능하나 변환 공식은 별도).
- **국내 국채 3년/10년 지표금리**: 지표물이 주기적으로 교체(로테이션)되어 과거일 정확 재현에 별도 처리 필요.
