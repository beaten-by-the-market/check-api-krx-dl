# 공시·뉴스 결합 분석 케이스 시나리오

이 폴더는 **KRX 공시**(유상증자·자기주식·최대주주변경·시장경보 등)와 **언론 뉴스**를
주가·수급(외국인/기관/개인)·공매도와 **결합**해, "그 공시나 뉴스가 실제로 주가에 무엇을
했는가"를 보는 시나리오들을 **도메인(자본시장) 관점**에서 정리한 것입니다.

코드나 API를 몰라도 됩니다. **그냥 한국어로 "이런 걸 보고 싶어"라고 말하면**, LLM이 알아서
공시·뉴스·주가·수급·공매도 데이터를 가져와 표와 해석을 만들어 줍니다. 이 문서들은 *무엇을
말하면 무엇이 나오는지*를 실제 예시(전부 실호출로 검증됨, 2026년 6~7월 기준)와 함께 보여줍니다.

## 어떻게 쓰나요?

1. 아래 표에서 하고 싶은 분석을 고릅니다.
2. 해당 문서를 열어 **"LLM에게 이렇게 말하세요"** 예시 문장을 참고합니다.
3. 그 문장을 (종목명·날짜만 바꿔서) 채팅에 그대로 입력합니다.
4. 나오는 결과를 **"결과 읽는 법"** 설명과 함께 해석합니다.

> 말투는 자유입니다. "가온전선 무상증자 공시 나고 주가 어떻게 됐어?"처럼 편하게 물어도 되고,
> 문서의 예시 문장을 복사해 종목만 바꿔도 됩니다. 정확한 용어를 몰라도 LLM이 되묻습니다.

## 알아둘 점 (딱 3가지)

이 데이터를 이해할 때 도움이 되는 최소한의 배경입니다. 몰라도 결과 해석엔 지장 없습니다.

1. **시장 구분은 자동입니다.** 종목이 코스피인지 코스닥인지 LLM이 알아서 판별해 맞는 시장의
   주가·수급·공매도 데이터를 가져옵니다. "가온전선은 코스피, 카카오게임즈는 코스닥" 같은 걸
   직접 지정할 필요가 없습니다.
2. **"공시"는 세 종류를 함께 봅니다.** ①기업공시(유상증자·자기주식·최대주주변경·배당·실적 등),
   ②시장경보(투자주의·투자경고·투자위험·단기과열·공매도과열), ③상장상태(상장·상장폐지 등).
   그래서 "회사가 스스로 낸 공시"와 "거래소가 붙인 경보"를 한 화면에서 볼 수 있습니다.
3. **뉴스에는 종목 태그가 없습니다.** 뉴스는 종목별로 직접 조회하는 방식이라, "이 뉴스가 어느
   종목 뉴스인지"를 데이터가 미리 달아두지 않습니다. 그래서 뉴스 분석은 항상 "특정 종목/키워드로
   찾아서" 진행합니다. → 자세한 건 [06 뉴스 커버리지·감성](06_뉴스커버리지감성.md).

## 한 가지 짚고 넘어갈 것 — 이벤트일(D0)의 정의

이 폴더에서 "D0(이벤트 당일)"은 **공시일 이상 첫 거래일**을 뜻합니다. 특히 시장경보류는
**장 마감 후 발표**되는 경우가 많아, 그날 종가에는 이미 반영이 끝나 있고 실제 시장 반응은
다음 거래일부터 나타납니다. 그래서 D0을 "공시 효과가 처음 주가에 실릴 수 있는 거래일"로 잡습니다.
