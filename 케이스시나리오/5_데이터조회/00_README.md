# 데이터 조회 기본 케이스 시나리오

이 폴더는 앞의 분석 유형들과 **성격이 다릅니다**. 어닝 서프라이즈·스크리너·시황 재현처럼
데이터를 *가공해 해석*하는 게 아니라, 그 재료가 되는 **원천 데이터를 그냥 가져오는**
가장 기본 기능입니다. 특정 분석이 아니라 "아무 종목·지표나 원하는 걸 뽑아줘"에 해당합니다.

코드나 API를 몰라도 됩니다. **그냥 한국어로 "삼성전자 6월 일별 시세 뽑아줘", "원/달러 환율
코드 뭐야?"처럼 말하면**, LLM이 알맞은 데이터 창구를 찾아 **표·파이썬 코드·CSV 중 원하는
형태로** 만들어 줍니다. 이 문서들은 *무엇을 말하면 무엇이 나오는지*를, 검증된 앵커 숫자와
결과 형태(스키마)로 보여줍니다.

## 어떻게 쓰나요?

1. 아래 표에서 하고 싶은 일(그냥 뽑기 / 코드부터 찾기)을 고릅니다.
2. 해당 문서를 열어 **"LLM에게 이렇게 말하세요"** 예시 문장을 참고합니다.
3. 그 문장을 (종목·지표·기간만 바꿔서) 채팅에 그대로 입력합니다.
4. 나오는 결과를 **"결과 읽는 법"** 설명과 함께 해석합니다.

> 말투는 자유입니다. "코스피 종합 지수값 뽑아줘"처럼 편하게 물어도 되고, 정확한 지표명이나
> 종목코드를 몰라도 됩니다. 애매하면 LLM이 되묻거나, 먼저 "어디서 보는지"부터 찾아줍니다.

## 알아둘 점 (딱 3가지)

이 유형을 이해할 때 도움이 되는 최소한의 배경입니다. 몰라도 결과 해석엔 지장 없습니다.

1. **커버리지가 넓습니다.** 개별종목 주식(시세·거래량·시총·외국인보유율)은 물론, 지수·ETF·
   채권 금리·파생(선물 미결제·베이시스)·원/달러 환율·공시·뉴스·재무·투자자 수급까지
   한 창구에서 뽑을 수 있습니다. "이런 것도 되나?" 싶으면 일단 물어보는 게 빠릅니다.
2. **산출물은 3종입니다.** ①바로 보는 **표**, ②재사용 가능한 **파이썬 코드**, ③**CSV 파일**.
   기본은 표이고, "코드로 줘" 또는 "CSV로 저장해줘"라고 덧붙이면 형태를 바꿔 줍니다.
3. **없는 데이터는 솔직히 알려줍니다.** 범위 밖인 지표는 "이건 여기선 안 나온다"고 명확히
   안내합니다. 다만 지표명으로 안 나온다고 바로 없는 건 아니어서(→ [시나리오 2](02_지표코드찾기.md)),
   코드 카탈로그와 필드까지 뒤진 뒤에 "없음"을 판정합니다.
4. **거래대금/거래량/순위는 거래소를 확인하세요.** "거래대금 상위 20" 같은 걸 그냥 뽑으면 기본은
   **통합(KRX+NXT)**입니다. KRX만/NXT만 원하면 말해 주세요. 단순 "일별 시세"의 종가·등락률은 KRX
   공식값이 기준입니다(NXT 출범으로 같은 종목이 두 거래소에서 거래됩니다).
