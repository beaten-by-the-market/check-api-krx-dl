# 시나리오 7. 재무 트렌드 — 몇 년간 실적과 마진이 어떻게 변했나

## 한 줄 요약
매출·영업이익·영업이익률(마진)의 **다년/다분기 추이**를 보여주고, 이어서 증권사의 **향후 추정**을
붙입니다. "이 회사 실적이 어느 사이클에 있나"를 한눈에 봅니다.

## 이게 답해주는 질문
- 이 회사 매출·영업이익이 몇 년간 어떻게 움직였나?
- 마진(영업이익률)이 좋아지고 있나 나빠지고 있나?
- 지금은 사이클의 바닥인가 고점인가?

## LLM에게 이렇게 말하세요
- "삼성전자 최근 5년 매출·영업이익·마진 추이 보여줘"
- "삼성전자 실적 트렌드 정리해줘 (분기별로)"
- "이 회사 영업이익률이 개선되고 있는지 몇 년치로 보여줘"
- "에코프로비엠 실적 사이클 보여줘"

## 그러면 이런 결과가 나옵니다
**삼성전자 분기 누적 추이 예시 (연결):**

| 회계연월 | 매출 | 영업이익 | 영업이익률 |
|---|---|---|---|
| 2021년 | 279.60조 | 51.63조 | 18.5% |
| 2022년 | 302.23조 | 43.38조 | 14.4% |
| 2023년 | 258.94조 | 6.57조 | **2.5%** ← 메모리 불황 |
| 2024년 | 300.87조 | 32.73조 | 10.9% |
| 2025년 | 333.61조 | 43.60조 | 13.1% ← 회복 |

**에코프로비엠 예시 (2차전지 붐-버스트가 선명):**
```
2021년  OPM 7.7%  →  2023년 초 8~9%(호황)  →  2024년 -1.3%(적자 전환)  →  2025년 5.7%(회복)
```

## 결과 읽는 법
- **마진(영업이익률)이 핵심**: 매출은 비슷해 보여도 마진이 급변합니다. 삼성전자는 2023년
  메모리 불황에 영업이익률이 **2.5%까지** 추락했다가 2025년 13.1%로 회복했습니다.
  → 반도체는 대표적인 사이클 산업이라는 게 숫자로 보입니다.
- **에코프로비엠**: 2차전지 소재 단일 사업이라 업황을 그대로 탑니다. 2023년 호황 → 2024년 적자 →
  2025년 회복의 붐-버스트가 마진에 선명합니다([시나리오 5 사업 프로파일](05_사업프로파일.md)의
  "양극재 97%"와 연결해서 보면 이해가 쉽습니다).
- **향후 추정**: 실제 과거치 뒤에 증권사 컨센 향후 전망이 붙습니다(합리적 범위인 경우만 표시).

## 이렇게도 바꿔 말할 수 있어요
- **기간 조절**: "최근 10년치로 보여줘", "분기 단독(누적 말고)으로 보여줘"
- **지표 추가**: "순이익이랑 ROE도 같이", "매출 성장률도 넣어줘"
- **비교**: "삼성전자랑 SK하이닉스 마진 추이 같이 보여줘"

## 주의사항
- **누적 vs 분기 단독**: 기본은 "누적"(예: 2025년 9월 = 1~9월 합)입니다. 분기별 단독 실적을
  원하면 "분기 단독으로"라고 말하세요.
- **먼 미래 추정치(2년 이상)는 부정확**할 수 있어, 비현실적인 값은 자동으로 걸러지거나 경고가
  붙습니다. 실제로 삼성전자의 2027년 이후 향후 컨센은 데이터가 부풀려져 자동 제외됩니다.
- **연결 기준**이 기본입니다.

## 뒤에서 쓰는 데이터 (참고)
- 과거 실제: IFRS 계정별 추이 (`/etc/ifrs/comp_ifrs_hist`)
- 향후 추정: 컨센서스 다년 (`/etc/cons/comp_cons_hist`, 비현실적 값 자동 필터)
- 실행 함수: `f.fundamental_trend("005930")`
