# 기업 펀더멘털 분석 케이스 시나리오

이 폴더는 **기업정보·재무(IFRS)·증권사 추정치(컨센서스)** 데이터를 활용해 할 수 있는
분석 시나리오들을 **도메인(자본시장) 관점**에서 정리한 것입니다.

코드나 API를 몰라도 됩니다. **그냥 한국어로 "이런 걸 보고 싶어"라고 말하면**, LLM이
알아서 데이터를 가져와 표와 해석을 만들어 줍니다. 이 문서들은 *무엇을 말하면 무엇이 나오는지*를
실제 예시(전부 실데이터로 검증됨, 2026년 7월 기준)와 함께 보여줍니다.

## 어떻게 쓰나요?

1. 아래 표에서 하고 싶은 분석을 고릅니다.
2. 해당 문서를 열어 **"LLM에게 이렇게 말하세요"** 예시 문장을 참고합니다.
3. 그 문장을 (종목명만 바꿔서) 채팅에 그대로 입력합니다.
4. 나오는 결과를 **"결과 읽는 법"** 설명과 함께 해석합니다.

> 말투는 자유입니다. "삼성전자 작년 실적 컨센 대비 어땠어?"처럼 편하게 물어도 되고,
> 문서의 예시 문장을 복사해 종목만 바꿔도 됩니다. 정확한 용어를 몰라도 LLM이 되묻습니다.

## 자주 쓰는 종목코드 (예시)

| 삼성전자 | 005930 | SK하이닉스 | 000660 |
| 에코프로비엠 | 247540 | 삼성SDI | 006400 |
| 현대차 | 005380 | 기아 | 000270 |
| NAVER | 035420 | 카카오 | 035720 |
| 한미반도체 | 042700 | POSCO홀딩스 | 005490 |

> 코드를 몰라도 종목명(예: "삼성전자")으로 말하면 됩니다. LLM이 코드로 변환합니다.

## 알아두면 좋은 배경 상식 (딱 3가지)

이 데이터를 이해할 때 도움이 되는 최소한의 개념입니다. 몰라도 결과 해석엔 지장 없습니다.

1. **연결 vs 별도** — 자회사까지 합친 게 "연결", 본사만 보는 게 "별도". 증권사 분석·밸류에이션은
   보통 **연결** 기준입니다. 이 문서의 모든 예시는 연결 기준입니다.
2. **컨센서스(컨센)** = 여러 증권사 애널리스트 예상치의 평균. "시장 눈높이"라고 보면 됩니다.
   이 데이터엔 예상치가 **날짜별로 쌓여 있어**, 시간이 지나며 눈높이가 어떻게 바뀌었는지(리비전)를
   그대로 볼 수 있습니다. → 시나리오 2·8·9의 핵심.
3. **회계연도(YYMM)** — "202512"는 2025년 12월 결산(=2025 연간)을 뜻합니다. 분기는 202503(1분기),
   202506(반기 누적) 식입니다.

## 두 가지 주의 (LLM이 알아서 걸러주지만 참고)

- **먼 미래 추정치는 부정확할 수 있습니다.** 당해/직전 연도 추정은 정확하지만, 2년 이상 먼 미래
  예측치는 데이터가 부풀려지는 경우가 있어, LLM이 자동으로 걸러내거나 경고를 붙입니다.
- **목표주가·투자의견은 이 데이터에서 제공되지 않습니다.** 대신 "전망치가 올랐나/내렸나"(리비전)로
  같은 신호를 대체합니다(시나리오 2).
