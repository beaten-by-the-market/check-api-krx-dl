# CHECK API / kquant MCP Knowledge Base

This repository contains a curated knowledge base for building an MCP server around KOSCOM CHECK API and the `kquant` Python package.

Start here:

- [docs/README.md](docs/README.md): main documentation index
- [docs/FOLDER_STRUCTURE.md](docs/FOLDER_STRUCTURE.md): folder guide
- [checkapi-specs.json](checkapi-specs.json): machine-readable CHECK API specs
- [kquant-docs.json](kquant-docs.json): machine-readable kquant function catalog
- [build-knowledge-docs.js](build-knowledge-docs.js): regenerates `docs/`

The final documentation includes endpoint catalogs, chatbot guidance, kquant mappings, and Python examples for loading data.
