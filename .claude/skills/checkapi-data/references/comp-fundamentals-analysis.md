# 기업정보·IFRS·컨센서스 입체 분석 (etc/comp · etc/ifrs · etc/cons)

CHECK API '기타-API'의 펀더멘털 3축을 **서로/시세·수급·공시 하네스와 결합**해 돌리는
시나리오 지도. 헬퍼는 `scripts/comp_fundamentals_lab.py`(등록 IP·샌드박스 밖 실행).
아래 수치·구조·함정은 전부 **실호출로 검증**(삼성전자 005930 / SK하이닉스 000660 /
에코프로비엠 247540, 2026-07 기준).

```python
import comp_fundamentals_lab as f     # scripts/ 에서 실행
f.earnings_surprise("005930","202512")            # CS1
f.estimate_revision("005930","202512","121500")   # CS2
f.earnings_event_study("005930","202512")         # CS3
f.peer_valuation(["005930","000660"],"202512")    # CS4
f.business_profile("005930","202512")             # CS5
f.ownership("005930","202412")                     # CS6
f.fundamental_trend("005930")                      # CS7
# 전체 데모:  python comp_fundamentals_lab.py demo 005930 202512
```

## 반드시 아는 구조 (실측)

### TERM_TYP / TERM_TYPE — 별도/연결 × 누적/분기
IFRS·컨센의 `TERM_TYP`, 실적발표일의 `TERM_TYPE`:
- **1x = 별도(개별), 3x = 연결.**  (삼성 FY2025 영업이익: 별도 23.6조=term1, 연결 43.6조=term31)
- **x1 = 누적**(YYMM까지 누계: 연간/반기/9M),  **x2 = 분기단독.**
- **애널리스트/밸류에이션 표준 = 연결 → 기본값 `CON="31"`(누적) / `"32"`(분기).** `SEP="1"`.

### 단위
- 금액계정(매출/영익/순익/자산…)은 **천원**.  `값÷1e9 = 조`,  `값÷1e5 = 억`.
- per-share(EPS·BPS·DPS)·목표주가는 **원**.
- 비율계정(ROE·OPM·순이익률·배당성향, 코드일람 `단위=7`)은 **소수분수** → ×100해야 %.
  (ROE 0.0999 = 9.99%).  배당수익률(`단위=6`)은 이미 %.  PER/PBR/PSR/EV_EBITDA(`단위=8`)는 배수.

### 컨센서스 = 리비전 히스토리 내장
`comp_cons*`의 각 행은 `CNS_DT`(생성일자)를 갖는다. 한 대상연도(YYMM) 지표에 대해
**생성일자별 추정치 스냅샷 수백 개**(삼성 FY2025 영업이익 561개, 2022~2026)가 쌓여 있어,
추정치가 시간에 따라 어떻게 상·하향됐는지(리비전)를 그대로 복원할 수 있다.

## 시나리오 (전부 실호출 검증)

- **CS1 어닝 서프라이즈** `earnings_surprise` — 발표일(perf_date) 직전 컨센 vs 실제(IFRS),
  브리지로 매핑해 서프라이즈%. *삼성 FY2025: 영익 컨센 42.0조→실제 43.6조(+3.7%), EPS +10.4%.*
- **CS2 추정 리비전 모멘텀** `estimate_revision` — 컨센 생성일자 시계열의 최근 1주/1개월/3개월 변화.
  *삼성 영익 추정: 3개월 31.0조→42.0조(+35.5%) 상향 → 어닝비트로 연결.* (카운트 계정 미제공 → 시계열로 대체)
- **CS3 실적발표 이벤트 스터디** `earnings_event_study` — perf_date 발표일 전후 주가·거래량·
  외국인/기관 순매수(시세·수급 하네스 결합) + 서프라이즈. *삼성 D0 +6.4%, 발표전 +11.1% 선반영→발표후 -3.6%(sell-the-news), 외국인 순매도.* KOSDAQ은 자동 m003.
- **CS4 피어 밸류에이션 횡단면** `peer_valuation` — comp_basic 업종 + 컨센 멀티플/수익성.
  *SK하이닉스 PER 14.4·ROE 43.5%(HBM) vs 삼성 PER 23.6·ROE 10% vs 한미반도체 PER 65.*
- **CS5 사업 프로파일** `business_profile` — 매출구성비 + 시장점유율 + 기업소개.
  *삼성: DX 56%·DS 39%; 점유율 DRAM 34%·TV 29%·스마트폰패널 43%. 에코프로비엠: 양극재 97%.*
- **CS6 지분구조/유통물량** `ownership` — 주주현황 → 지배주주블록 지분·추정유통.
  *삼성 지배블록 20.07%(삼성생명 8.51·국민연금 7.25·BlackRock 5.03) → 유통 79.9%. 에코프로비엠 50.3%.*
- **CS7 다년 재무 트렌드** `fundamental_trend` — IFRS_hist 매출/영익/마진(실제) + 컨센 향후추정.
  *삼성 2023 메모리불황 OPM 1~2.5%→회복. 에코프로비엠 EV붐버스트 OPM 8.9%→-1.3%→5.7%.*
- **CS8 추정 상향 스크리너(횡단면)** `revision_screen` — 유니버스(시총상위) 각 종목의 당해예측연도
  컨센 최근 1개월 리비전% 랭킹 = "최근 추정치 상향된 종목". *실측(2026-07, FY2026 영익): 삼성SDI
  +33%(턴어라운드)·SK +14.5·SK하이닉스 +6.5·삼성전자 +5.7 — 반도체/배터리 업사이클과 일치.*
  ⚠ **절대레벨은 원거리 팽창으로 부정확 → 리비전%만 신뢰.** 추정≈0(턴어라운드)은 %가 튀어 `turnaround` 플래그.
- **CS9 추정 상향 이벤트 스터디** `revision_event_study` / `upgraded_then_reaction` — 과거 컨센 상향
  이벤트(시계열에서 win 스냅샷 누적 ≥ up_thresh%) 탐지 → 이벤트일 D0 대비 +1/5/20거래일 주가반응(시세 결합).
  `upgraded_then_reaction`은 CS8(최근 상향 상위)에 각 종목의 과거 상향→주가반응을 붙인다.
  *실측(FY2025 영익 기준 +20D): SK스퀘어 +11.4%(승률71%)·포스코퓨처엠 +9.6%·삼성전자 +3.0%(70%) vs
  고려아연 -13.2%(17%)·삼성SDI -7.4% — 종목별로 '상향=매수신호' 유효성이 갈림.*
  주의: 이벤트는 한 대상연도(기본 202512) 리비전사에서 뽑아 2022~2025 분포, 이벤트 적으면(≤6) 신뢰 낮음.

크로스 아이디어: CS1×CS3(서프라이즈→주가반응 회귀), CS2×시세(추정상향 vs 주가 선후행),
CS4+CS7(피어 대비 마진/성장), perf_date→**실적발표 캘린더**(단, 전방 지평선 ≈ 직전분기까지).

## 함정 (실측으로 데인 것)

1. **원거리 향후 컨센은 신뢰 불가.** 당해/직전연도 추정은 정확하나, 2년+ 미래 대상연도는
   최근 스냅샷에서 값이 팽창한다(삼성 FY2026 영익 term31이 17조→**373조**로 표류, 분기 term32도 120조).
   `fundamental_trend`는 직전 실제치 대비 **0.3~3.0배 sanity gate**로 거른다(에코프로비엠 등 합리적 추정은 통과).
2. **애널리스트-오피니언 블록 미제공.** 코드일람엔 있으나 `comp_cons_item/hist(jcode+yyyymm+icode)`로는
   **빈 결과**다: 목표주가·투자의견·참여증권사(610xxx), 추정 상/하향 브로커수(235xxx·311xxx),
   Fwd.12M 롤링(121560·122760·312060·382160). 이 피드는 "회계연도 앵커 추정치"만 서빙 →
   목표주가 상승여력·리비전 카운트는 못 준다(리비전은 CS2처럼 추정치 시계열로 대체).
3. **응답 키 ≠ 명세 라벨(오타 주의).** 주주현황 보유비율은 `BB_COMM_RATE`(언더스코어, 명세는 BB_COMMRATE).
   실적발표일 기간구분은 `TERM_TYPE`(E, 다른 곳은 TERM_TYP). raw 키로 접근할 것.
4. **주주현황 `BB_SEQ=1`은 소계행**('대주주 1인 총계', BB_GUBN=10) → 개별과 합산하면 이중계상.
   `ownership`은 이 행을 지배지분으로 쓰고 개별은 따로 나열.
5. **한글 텍스트 깨짐은 표시만.** `call()`은 UTF-8 정상 반환. 콘솔 출력 시 `sys.stdout.reconfigure(encoding="utf-8")`
   (파일 저장은 utf-8-sig). 데이터 자체는 안 깨진다.
6. **comp_cons(yyyymm) 전체는 수만 행**(삼성 FY2025 68,797). 한 지표 리비전은 `comp_cons_item`이 가볍다.
7. **rank류 시세결합은 news_gongsi_lab 규칙 그대로** — 시장 라우팅 m001(KOSPI)/m003(KOSDAQ),
   투자자번호(외국인 F06508_11·기관 _08). CS3는 `pick_fam`으로 자동 폴백.
